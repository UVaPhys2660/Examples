About the files in this directory

printf_formats.cpp  : examples of using format specifiers 
functions.cpp       : examples of user defined functions
simple_loops.cpp    : examples of various loop constructs

make_test_data.cpp  : creates a data file "out.dat"

demo.plt            : gnuplot commands to plot data in "out.dat"
                      you can cut/paste these commands into gnuplot
                      or in gnuplot you can use 
  		           load "demo.plt"
                      to execute the commnds in the file
