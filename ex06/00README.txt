The program metals.cpp is a solution to Brooks Chapter 7, Problem 5.
This program shows how to use an array of structures and also 
demonstrates the usage of several string functions.

string.c demonstrates the use of several string manipulation functions.
See the source code for details.

Refer to the documentation for string.h and ctype.h for a more complete 
listing.  See section "Standard C String &  Character" in the C/C++ reference.

readfile.cpp : demonstrates how you may use fgetc to read one character 
at a time from a file.
